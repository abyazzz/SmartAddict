TENTANG MODEL DETEKSI

ALGORITMA & DATA METRIKS SISTEM DETEKSI DINI SMARTADDICT MACHINE LEARNING

🌳
Decision Tree
SUPERVISED • CLASSIFICATION

Algoritma klasifikasi terstruktur yang memecah kumpulan data menjadi sub-himpunan keputusan berdasarkan parameter fitur yang paling optimal dan informatif.

  • Struktur klasifikasi linear & transparan
  • Eksekusi prediksi berkecepatan tinggi
  • Akurat untuk pemisahan data linear
📍
K-Nearest Neighbors
INSTANCE-BASED • NEIGHBORING

Menentukan klasifikasi koordinat data baru berdasarkan kalkulasi spasial jarak spasial (Euclidean) terdekat dengan kelompok data latih yang ada.

  • Responsif terhadap perubahan parameter baru
  • Efektif untuk kurva data non-linear
  • Arsitektur klasterisasi sederhana & kokoh
🧠
Neural Network
DEEP LEARNING • MULTILAYER PERCEPTRON

Arsitektur kognitif berlapis (input, hidden, output) yang memproses pembobotan data dinamis guna mengenali korelasi pola data kecanduan yang sangat kompleks.

  • Kemampuan adaptasi parameter tingkat tinggi
  • Unggul dalam memetakan anomali non-linear
  • Sangat akurat dengan parameter masukan dinamis
⚔️
Support Vector Machine
KERNEL METHOD • MAXIMUM MARGIN

Memproyeksikan data ke ruang berdimensi tinggi menggunakan kernel optimal guna memisahkan kategori kecanduan dengan margin hyperplane terlebar.

  • Sangat tangguh terhadap risiko overfitting
  • Akurasi maksimal pada skala data terlokalisasi
  • Kalkulasi margin pemisahan kelas presisi tinggi
